Spam detection SVM



This project was created with the goal of classify string inputs in two groups Spam or Not Spam (Ham).
I used two different classification algorithms to find out wich one perform better, Support vector machine or random forest. Support vector machines was superior for 1% having 95% of accuracy.

Limitations

The dataset used for this project was really small having just 5.572 observations, but once cleaned just 1.494 observations. This means the algorithm wont be as accurate as needed because still need more observations to get better doing this task.
