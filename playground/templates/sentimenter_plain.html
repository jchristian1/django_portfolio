Sentiment Analysis NLTK



This project was created to see how to get a sentiment analysis based on an input string. This project is using vader to accomplish the goal.
The VADER Sentiment Analyzer uses a lexical approach. That means it uses words or vocabularies that have been assigned predetermined scores as positive or negative. The scores are based on a pre-trained model labeled as such by human reviewers.

Limitations

The limitations of this pre-trained algorithm include old technologies (Not using transformers), using an architecture that are not the most efficient way to solve the problem and others.
